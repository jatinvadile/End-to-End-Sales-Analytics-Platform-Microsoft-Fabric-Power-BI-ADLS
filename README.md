# End-to-End_Sales_Analytics_Platform_Microsoft-Fabric_Power-BI_ADLS
## Tech Stack: Microsoft Fabric, Power BI, ADLS, Lakehouse, DAX, PySpark, Data Pipelines

1. Architected and implemented an end-to-end analytics platform on Microsoft Fabric, using Bronze-Silver-Gold Medallion architecture to transform raw source data into curated, business-ready datasets.
2. Built and orchestrated automated data ingestion and transformation pipelines from ADLS to Lakehouse, leveraging Fabric Pipelines and notebooks to support scalable and reliable data processing.
3. Designed a star schema semantic model with fact and dimension tables to enable high-performance analytical querying and downstream reporting.
4. Developed optimized DAX measures and KPI logic for revenue, margin, order value, quantity, and growth analysis, improving report responsiveness and analytical usability.
5. Automated the full data-to-report workflow, including pipeline orchestration and semantic model refresh, reducing manual reporting effort by approximately 65% and improving data refresh efficiency by approximately 50%.
6. Built interactive Power BI dashboards with drill-down analysis, slicer-driven filtering, and business KPIs, enabling stakeholders to make decisions 2–3x faster through near real-time reporting.
7. Improved data consistency and trustworthiness by implementing structured transformation layers and centralized Lakehouse storage, reducing reporting discrepancies by approximately 40%.
8. Enhanced report performance by approximately 30% through dimensional modeling, aggregation design, and optimized calculation logic.
